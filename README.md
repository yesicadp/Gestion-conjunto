# Gestión de Conjuntos Residenciales

Sistema web para apoyar la administración de conjuntos residenciales.

## Funcionalidades principales
- Gestión de apartamentos y residentes
- Registro de pagos de administración
- Control de parqueaderos
- Reservas de zonas comunes
- Publicación de anuncios

## Roles del sistema

### Administrador
- Registrar apartamentos y residentes
- Registrar pagos
- Gestionar parqueaderos
- Aprobar reservas
- Publicar anuncios

### Residente
- Ver estado de pagos
- Solicitar reservas
- Ver parqueadero asignado
- Ver anuncios

- HTML
- CSS

- SQLite / PostgreSQL
