# Visión del Proyecto

## Problema
La administración de conjuntos residenciales muchas veces se maneja por WhatsApp o Excel,
lo que genera desorden y falta de control sobre pagos, parqueaderos y reservas.

## Solución
Desarrollar una plataforma web que permita centralizar la administración
del conjunto y facilitar la comunicación entre administradores y residentes.

## Usuarios

Administrador del conjunto:
- Gestiona pagos
- Gestiona parqueaderos
- Administra reservas

Residente:
- Consulta su estado de pagos
- Solicita reservas
- Consulta anuncios
